# Project Blueprint - Lotto Number Generator

## Overview
A web-based Lotto Number Generator with a modern UI, supporting both Dark and Light modes. Built with pure HTML, CSS, and JavaScript using Web Components.

## Features implemented
- **Lotto Number Generation**: Generates 6 unique numbers (1-45) with sorting and animated display.
- **Web Components**: Custom `<lotto-ball>` element for encapsulated styling and logic.
- **Modern CSS**: Responsive layout, CSS variables for theming, and smooth transitions.
- **Theme Support**: Dark mode and Light mode functionality with persistent settings.
